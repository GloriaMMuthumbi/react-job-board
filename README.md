# React Job Board

A responsive job board application built with React and Tailwind CSS. Users can browse job listings, filter by job type, and explore opportunities through a clean modern interface.

## Features
- Responsive design
- Job cards
- Filtering system
- Reusable React components
- Clean UI with Tailwind CSS

## Tech Stack
- React
- Tailwind CSS
- Vite

## Installation

```bash
npm install
npm run dev
